What is the document object model (DOM) ?

The DOM (Document Object Model) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each object corresponds to a part of the document (like elements, attributes, or text). The DOM allows programming languages, like JavaScript, to interact with and manipulate web pages dynamically.

what is event delegation in the contex of the dom and why it useful ?

Event Delegation is a technique in the context of the Document Object Model (DOM) that allows you to manage events more efficiently by utilizing the bubbling phase of event propagation. Instead of attaching an event listener to each individual child element, you attach a single event listener to a common parent element. This parent listens for events that bubble up from its children.
